HTML stands for Hyper Text Markup Language, which is the core language used to structure content on the web. It organizes text, images, links, and media using tags and elements that browsers can interpret.

The term “HyperText” refers to the ability to link different web pages using hyperlinks, allowing users to navigate easily between documents.“Markup Language” means it uses tags to define how text, images, and other elements are displayed on a webpage.

HTML was first developed by Tim Berners-Lee in 1991 while working at CERN.It has evolved through many versions — from HTML 1.0 to the latest HTML5, which was officially released in 2014.Each version has introduced new features to support multimedia, form handling, and modern web design.

An HTML document is made up of tags enclosed in angle brackets < >.A basic HTML page structure includes , , and tags.The contains meta information, title, and links to stylesheets, while the contains the visible content of the webpage.

HTML works with CSS media queries to make websites responsive, meaning they adapt to different screen sizes like mobiles, tablets, and desktops.This helps in creating user-friendly interfaces for all devices.
